## ✨ 功能特点

| 功能 | 描述 | 状态 |
|------|------|------|
| 📥 **批量导入存档** | 一键将下载的 ZIP 地图批量解压到 `.minecraft/saves` 文件夹 | ✅ 已完成 |
| 📤 **导出备份** | 将现有存档打包备份 | 🚧 开发中 |
| 📋 **查看列表** | 管理你的所有存档 | ✅ 已完成 |
| 🔧 **存档修复** | 修复损坏的存档文件 | 📋 待开发 |
| ℹ️ **关于软件** | 显示软件信息和更新日志 | 📋 待开发 |
| ❤️ **赞助一下** | 支持开发者 | ✅ 已完成 |
