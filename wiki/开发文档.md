## 代码架构

### 模块职责

| 文件 | 职责 |
|------|------|
| `main.py` | 程序入口，创建 App 实例并启动主循环 |
| `app.py` | 主应用类，包含所有 GUI 界面和业务逻辑 |
| `utils.py` | 工具函数（文件操作、图像处理等） |
| `config.py` | 配置常量、路径管理、全局导入 |
| `message.py` | 消息对话框组件 |

### App 类结构

```python
class App:
    # 1. 初始化方法
    def __init__(self)
    
    # 2. UI 创建方法
    def create_header()
    def create_buttons()
    
    # 3. 核心功能方法
    def import_save()      # 导入存档
    def export_save()      # 导出存档
    def list_saves()       # 存档列表
    def donate()           # 赞助
    def about()            # 关于
    
    # 4. 辅助功能方法（私有）
    def _select_version_saves()   # 选择版本
    def _show_donate_qr()         # 显示赞助码
    def _progress_window()        # 进度窗口
```

## 开发规范

### 代码风格

- **语言**：代码和注释使用中文
- **命名**：使用英文变量名和函数名，遵循 snake_case
- **文档**：所有函数使用中文文档字符串
- **导入**：在 `config.py` 中集中管理导入

### 函数注释格式

```python
def function_name(param: type) -> return_type:
    """函数简短描述

    Args:
        param: 参数说明

    Returns:
        返回值说明
    """
```

### 辅助方法命名

内部使用的辅助方法使用下划线前缀：

```python
def _helper_method(self):
    """这是内部辅助方法"""
    pass
```

## 构建发布

### 开发环境运行

```bash
python src/main.py
```

### 打包可执行文件

```bash
cd src
pyinstaller -F --noconsole --icon="../icon.ico" --name="存档管理器" --add-data="../img;img" --add-data="../icon;icon" --add-data="../fonts;fonts" --add-data="../sounds;sounds" main.py
```

参数说明：
- `-F`: 打包成单个可执行文件
- `--noconsole`: 隐藏控制台窗口
- `--icon`: 应用程序图标
- `--name`: 可执行文件名称
- `--add-data`: 包含资源文件夹（img、icon、fonts、sounds）

构建后的文件在 `dist/` 目录中。
